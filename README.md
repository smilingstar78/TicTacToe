## 🧠 AI ENGINE (Core Intelligence)

This AI uses the Minimax algorithm to simulate all possible game outcomes.
It plays optimally by assuming the opponent also plays perfectly.

---

## ⚙️ How the AI Thinks:

- 📊 Builds a full game tree
- 🔁 Recursively evaluates all moves
- 🎯 Assigns utility scores: <br>
Win → +10 <br>
Loss → -10 <br>
Draw → 0
- ⚡ Uses Alpha-Beta pruning to optimize performance

## 🧩 HOW THE AI WORKS:

<div style="background:#0f172a;padding:15px;border-radius:10px;color:#fff">

1️⃣ Game state is evaluated as a tree <br>
2️⃣ AI simulates all possible moves <br>
3️⃣ Opponent moves are assumed optimal <br>
4️⃣ Scores are assigned to terminal states <br>
5️⃣ Best move is selected using minimax <br>
6️⃣ Alpha-beta pruning removes unnecessary branches ⚡ <br>

</div>
